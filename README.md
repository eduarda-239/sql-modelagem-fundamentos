# sql-modelagem-fundamentos

Este repositório reúne estudos iniciais sobre SQL e modelagem de banco de dados relacional,
com foco em análise de dados e Business Intelligence.

## Conteúdos abordados
- Conceitos de bancos de dados relacionais
- Modelagem relacional
- Criação de tabelas com CREATE TABLE
- Chaves primárias e estrangeiras
- Inserção e manipulação de dados (INSERT, UPDATE, DELETE)
- Consultas com SELECT
- Filtros (WHERE, LIKE, DISTINCT)
- Ordenação de dados (ORDER BY)
- Junções entre tabelas (INNER JOIN)

## Ferramentas utilizadas
- MySQL
- MySQL Workbench
- UniController (Star Schema)

## Objetivo
Construir uma base sólida em SQL para análise de dados, visando atuação como Analista de BI Júnior.
